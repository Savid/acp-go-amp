# Go API

Import the module as:

```go
import ampacp "github.com/savid/acp-go-amp"
```

Primary entry points:

- `ampacp.NewAgent(opts ...Option) *Agent`
- `ampacp.Serve(ctx, input, output, opts ...Option) error`

Common options:

- `WithExecutablePath`
- `WithHome`, which is a parent for isolated per-session native `HOME` and XDG
  directories
- `WithDefaultModel`
- `WithEnv`
- `WithSessionStore`
- `WithConcurrencyLimits`
- `WithSeedFiles`, which maps relative paths to file contents written into each
  session's isolated native `HOME` before amp launches. See
  [Seed files](/features/models-config#seed-files). Amp is locked to its hosted
  backend, so this common option exists for family symmetry and cannot repoint
  Amp at a custom LLM endpoint.
- `WithTurnTimeout`, an optional per-turn native deadline (default `0`, no
  deadline); on expiry the turn fails with `cause: "timeout"`
- OpenTelemetry provider options

Amp per-session options:

- `AmpOptions.Env`
- `AmpOptions.Mode`
- `AmpOptions.Effort`
- `AmpOptions.Model` and `AmpOptions.OutputSchema`, which are accepted in the
  struct for API symmetry but rejected when supplied.

Request helpers include builders for new/load/resume/delete, text prompts,
config-option updates, stdio MCP servers, and HTTP MCP servers.
